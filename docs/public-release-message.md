# Public Release Message

SplatVault is a local-first multimodal asset search workspace for teams managing large image, video, scan, and project-file libraries.

It helps you find assets by meaning, not just by filename.

## What It Solves

- messy folder structures
- long videos that are painful to browse manually
- scattered project notes and media assets
- slow reuse of historical material
- weak pre-training and pre-production curation workflows

## What It Is

- local-first
- single-user preview edition
- public community foundation for a future Pro line

## What It Is Not

- not a generic AI chatbot
- not a 3D Gaussian Splatting renderer
- not an enterprise deployment
- not a finished Pro product yet

## Public Positioning

- Community Edition is the public discovery layer
- Pro features will be split later
- binaries and premium packaging will be delivered separately

## Short Pitch

`Search your local asset library by meaning, not filenames.`
