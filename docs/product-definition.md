# SplatVault 产品定义

## 项目名称
SplatVault

## 核心用户
- 3D/视频/扫描团队的技术美术、制片人、项目经理
- 需要管理大量素材库的数据科学家、AI训练团队
- 需要快速复用历史项目资产的创意工作室

## 核心场景
1. 找参考图 - 文件夹层级混乱，用自然语言描述搜索
2. 视频片段定位 - 长视频手动翻找耗时，关键帧+语义索引
3. 项目交接 - 资产与上下文文档分离，项目空间绑定
4. 训练数据准备 - 图片视频分散，集合筛选+批量导出

## 核心卖点
- 语义搜索：不用记文件名，用自然语言描述内容
- 视频可搜：不只是文件名，片段内容也能匹配
- 工作台整合：搜索、筛选、归档、导出在一个界面

## MVP 边界
✅ 本地文件导入（目录扫描）
✅ 图片/视频/文本三类资产
✅ 视频关键帧提取
✅ 文本搜图、文本搜视频、以图搜图
✅ 项目/集合/标签三级组织
✅ Web 工作台（搜索+预览+筛选）

## 非目标
❌ 不做 3DGS 渲染或点云处理
❌ 不做图像压缩/转码优化
❌ 不做聊天机器人或AI助手
❌ 不做"量子""意识""奇门"概念包装
❌ 不做多用户协作（先单机版）
