# SplatVault Public Preview

SplatVault is a local-first multimodal asset search workspace for teams managing image, video, scan, and project-file libraries.

This preview release is focused on the core workflow:

- import local assets
- organize them by project, collection, and tag
- search across assets faster than folder-by-folder browsing
- prepare a stronger base for future semantic and video-centric retrieval

## Who This Preview Is For

- 3D and scan-heavy teams
- video production teams
- technical artists
- AI/data teams preparing training material
- studios with large local asset libraries

## What Works In This Preview

- local project structure
- image/video/text asset import foundation
- thumbnail and preview workflow
- search workspace foundation
- collection and tag workflows
- batch-oriented product direction for future retrieval workflows

## What This Preview Is Not Yet

- not a polished multi-user platform
- not an enterprise deployment
- not a finished Pro release
- not a 3D Gaussian Splatting renderer
